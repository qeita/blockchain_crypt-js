# blockchain_crypto-js  
  
Refer to:  
[javascriptでブロックチェーンをつくってみよう](https://mizumotok.hatenablog.jp/entry/2018/04/21/152341)  

Environment:  
- node `v10.9.0`  
- yarn    

Set up:  
  
1. Install packages  
$ `yarn`  
  
2. Initialize flow  
$ `yarn flow init`  
  
3. Run test with jest  
$ `yarn flow-typed install jest@v22.x.x`
